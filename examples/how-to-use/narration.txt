Script to Video. The fastest way to turn a narration script and slides into a polished, narrated video. No video editing software needed. Works as a portable skill for any AI coding agent.

The absolute easiest way to use this tool is through an AI agent. Whether you use Copilot, Claude, or Cursor, just add the skill file to your repo and your agent can build videos for you. Run mkdir dash p dot github slash skills slash video, then curl the skill file from the script to video repo. Once the skill is in your project, your agent knows every command, every voice, every transition, and the from doc workflow.

Here is what an agent does when you ask it to create a video. It runs npx s2v from doc with your Google Doc URL and a slides folder. That single command pulls the narration, copies the screenshots, writes the config, and scaffolds the project. Then it runs npx s2v build and you have a finished video. No checkout, no install, just npx.

Authentication uses your existing Google credentials. Run g cloud auth application default login once, which opens your browser to sign in. After that, all Google Doc, Google Slides, and Drive access works automatically. No public sharing needed, no service account setup for local use.

You can also point to a Google Slides presentation or a PowerPoint file instead of individual screenshots. Set slides source to google slides or pptx in your demo dot yaml, and the tool exports each slide as an image automatically. All authenticated through the same credentials.

For the manual route without an agent, run npx s2v init to scaffold a project. This gives you a demo dot yaml config, a narration dot txt template, and a slides folder. Drop your images in, write your narration with one paragraph per slide, and run npx s2v build. That is the entire workflow.

Now let's talk about voices. Run npx s2v voices to browse the curated English voices. Ryan Neural is the default British male for professional demos. Jenny Neural is an American female for training. Guy Neural is a casual American male for tutorials. Each voice is tunable. Set rate to minus ten percent for slower delivery or plus twenty percent for faster. Adjust pitch for deeper or higher tones.

Slide transitions control how slides move. The default is a cross fade with over forty-five options available. Wipe left works for sequential content. Dissolve gives a subtle professional feel. Set it in demo dot yaml or with dash dash transition on the command line. Run npx s2v transitions to see the full gallery.

Background music is built in with six presets generated using audio synthesis. No licence concerns. Ambient is a warm pad for demos. Upbeat has energy for launches. Corporate is neutral for internal videos. Run npx s2v music to preview them. Or bring your own file with dash dash music path to your track.

Run npx s2v browse to open a local web page where you can preview every voice and music preset before committing. Click play on any voice to hear it with your chosen rate and pitch. Click preview on any music preset to hear a fifteen-second sample. This saves time experimenting.

To recap, the simplest path is to add the skill file to your repo and let your AI agent handle everything. It runs npx s2v from doc with a Google Doc URL, builds the video, and you are done. For manual use, npx s2v init, write narration, add slides, npx s2v build. Either way, polished narrated video in minutes.
