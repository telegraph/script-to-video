Version three point two is here. This release includes four major features, twelve bug fixes, and several performance improvements. Let's walk through the highlights.

First, the new search experience. We have completely rebuilt the search bar with fuzzy matching and instant results. Type any keyword and see suggestions appear in real time. Search results now include previews so you can find what you need without clicking through multiple pages.

Second, dark mode. The entire application now supports a system-aware dark theme. It follows your operating system preference by default, or you can toggle it manually from the settings menu. Every component, chart, and table has been tested in both light and dark modes.

Third, bulk actions. You can now select multiple items and perform actions on all of them at once. Select rows using the checkboxes, then choose an action from the toolbar. Supported actions include assign, label, archive, and export. This saves significant time for managers working with large backlogs.

Fourth, the new API version two. We have released a new version of the public API with improved rate limiting, better error messages, and webhook support. The old API continues to work but will be deprecated in six months. See the migration guide for details on upgrading.

On the performance side, page load times are down by forty percent across the board. We achieved this through lazy loading, query optimisation, and a new caching layer. The dashboard now loads in under two seconds on a standard connection.

That wraps up version three point two. Update today to get all these improvements. Check the changelog for the full list of changes, and reach out in the feedback channel if you run into any issues.
